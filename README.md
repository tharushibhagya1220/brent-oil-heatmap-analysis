# brent-oil-heatmap-analysis
# 🌍 Global Events vs Brent Oil Prices (2019–2026)

This project analyzes how major global events influenced Brent crude oil prices using a heatmap visualization.

## 📊 Overview

The heatmap shows monthly Brent crude oil prices (USD/barrel) across key global phases:

- Pre-Pandemic Baseline  
- COVID-19 Collapse  
- Recovery & Surge  
- Ukraine War Shock  
- Stabilisation  
- Gaza-Israel Conflict  
- Pre-Iran Escalation  
- US-Iran War Conflict  

Grey cells represent months where data is not available.

---

## 🔍 Key Insights

- 📉 Oil prices dropped sharply during COVID-19  
- 📈 Strong recovery in 2021  
- 🔥 Price spikes during geopolitical conflicts  
- ⚖️ Continued volatility in recent years
